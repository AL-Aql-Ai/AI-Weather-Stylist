**AI-powered outfit recommendations based on real-time weather, body metrics, and skin undertone analysis.**

## ✨ Features

- 🌤️ **Real-time Weather Integration** - Fetches live weather data for your city
- 🧬 **Body Type Analysis** - Personalized styling for ectomorph, mesomorph, endomorph
- 🎨 **Skin Undertone Matching** - Color palette recommendations based on warm/cool/neutral
- 🤖 **Groq AI-Powered** - Intelligent outfit generation using Llama 3.1
- 📊 **BMI Integration** - Health-aware fashion suggestions
- 💧 **Hydration Consideration** - Comfort-based fabric recommendations
- 📱 **Responsive UI** - Works on desktop and mobile

## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- OpenWeather API Key ([Get Free Key](https://openweathermap.org/api))
- Groq API Key ([Get Free Key](https://console.groq.com))
- Go to the settings and make the theme 'light'.
**Tech Stack**

Frontend: Streamlit

AI/ML: Groq API (Llama 3.1 8B)

Weather: OpenWeatherMap API

Styling: Custom CSS

### Installation

```bash
# Clone repository
git clone https://github.com/yourusername/ai-weather-stylist.git
cd ai-weather-stylist

# Install dependencies
pip install -r requirements.txt

# Run app
streamlit run app.py
